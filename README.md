# IntegraPNCP — Site institucional

Landing page do **IntegraPNCP**, sistema de gestão de informações para integração de contratações públicas com o [Portal Nacional de Contratações Públicas (PNCP)](https://www.gov.br/pncp).

**Produção:** [https://www.integrapncp.com.br](https://www.integrapncp.com.br)

## Stack

- Next.js 15 (export estático)
- TypeScript
- Tailwind CSS v4
- GitHub Pages + GitHub Actions

## Desenvolvimento

```bash
npm install
cp .env.example .env.local
npm run dev
```
